Individual submissions 🧋

Build your own personal website using HTML & CSS, host it on GitHub Pages, and submit it here to get $5 for boba. Individual submissions are open to US-based participants aged 13–18.

Never built a site before?

No problem at all, that's exactly who this is for! You can follow our personal website guide to build one from scratch. It walks through everything step by step.

Want to code with others? Join other high schoolers online who are building their first websites.
